**注意：**
1. 标准化API，模型的主要功能应该由模型中的方法完成，可以重复被调用，而不应该需要在Test方法如main中还需要进行进一步处理，所有的处理应该都在API中完成。
2. API中声明的静态变量若需声明则一定要初始化，否则在调用API时静态变量无初始值，不推荐使用静态，而应该使用对象变量，由调用者赋值。

####优化

2. 递归优化是对递归的次数进行优化。
